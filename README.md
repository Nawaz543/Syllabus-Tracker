# 📚 Syllabus Tracker

A modern and responsive **Syllabus Tracking Web App** made using pure **HTML, CSS, and JavaScript** in a **single file**.

Track your courses, subjects, chapter completion, and notes easily — with automatic local saving and dark/light mode support.

---

# ✨ Features

* 📖 Multiple Courses Support
* 📚 Unlimited Subjects & Chapters
* ✅ Chapter Completion Tracking
* 📊 Live Progress Bar
* 📝 Notes / Comment Box
* 💾 Auto Save using LocalStorage
* 🌙 Light / Dark Mode
* 📱 Fully Responsive UI
* ⚡ Smooth Animations
* 🔒 No Internet Required
* 📂 Single File Project

---

# 🖼️ How It Works

### 1. Open the file

When you open the file for the first time, a form appears.

### 2. Enter Course Details

Example:

* Course Name → `BTech 4th Sem`
* Subject → `Discrete Mathematics`
* Chapters → `5`

You can add multiple subjects and even multiple courses.

### 3. Track Progress

* Tick completed chapters
* Progress bar updates automatically
* Add personal notes/comments

### 4. Data is Saved Automatically

All data is stored in your browser using **LocalStorage**.

So even after:

* Closing browser
* Refreshing page
* Reopening file

your data remains safe ✅

# ⚠️ Important Note

Because this project uses **LocalStorage**, your data is saved only in the same browser/device.

If you:

* clear browser storage
* use another browser
* reset browser data

then saved syllabus data may be removed.
